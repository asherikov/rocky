Detailed Description

Geometric shape data. This class holds one or more lists of points. How these points are interpreted depends on the geometry Type.

Types of geometry supported in this class.

Enumerator
Points 

A single collection of points (in "points")

LineString 

A single collection of line segments (in "points")

Polygon 

One polygon. "points" contains the outer ring with a CCW winding. "parts" contains an optional collection of holes, each with a CW winding.

MultiPoints 

Each part contains one or more points; "points" is ignored.

MultiLineString 

Each part contains a separate linestring; "points" is ignored.

MultiPolygon 

Each part contains a separate polygon; "points" is ignored.
